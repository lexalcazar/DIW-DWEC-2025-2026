7. Hojas de estilo alternativas (modo claro/oscuro)

En este apartado debes crear dos hojas de estilos: una para modo claro y otra para modo oscuro. Ya tienes enlazada styles.css en la cabecera del documento. Añade también la hoja alternativa dark.css siguiendo las instrucciones de la tarea.

8. Validación del CSS

En este apartado deberás validar tu archivo styles.css usando el validador oficial de W3C. No es necesario modificar el HTML para este ejercicio, pero aquí puedes dejar anotaciones si lo deseas.

9. Preprocesador SCSS

En este apartado trabajarás en el archivo styles.scss. No es necesario modificar el HTML, pero puedes añadir aquí comentarios sobre las variables, anidación y mixins que has creado.
